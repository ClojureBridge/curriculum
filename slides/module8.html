Creating a project

Type the following in your terminal:

lein new quil drawing-demo
  • drawing-demo is the name of your project, it can be whatever you want.

lein - Leiningen

  • tool you run in your terminal

Anatomy of a Quil App

Most basic drawing needs only 2 functions defined:

  • (defn setup [])
  • (defn draw [])

Open src/drawing-demo/sample.clj.

Setting up

(defn setup []
  ;; Set frame rate to 30 frames per second.
  (q/frame-rate 30)
  ;; Set the stroke color
  (q/stroke 255 0 0))

Draw Example

(defn draw []
  (q/line 50 50 
          100 100))
  • (q/line start-x start-y end-x end-y)

Creating a sketch

(q/defsketch drawing-demo
  :title "name of my drawing"  ;; title of window
  :size [500 500]  ;; size of canvas
  :setup setup  ;;name of setup method
  :draw draw)   ;;name of draw method
